# 题目 18
数学 选择题 难度: 2/5
若式子 $sqrt{1-x}$ 在实数范围内有意义，则$x$的取值范围是（ ）A. $x>1$ B. $x<1$ C. $x geq 1$ D. $x leq 1$
🖼️ 题目附图
📐 图形描述
✓ 正确答案
D. $x leq 1$
✗ 你的答案
C. $x geq 1$
💡 详细解析
二次根式有意义的条件是其被开方数必须大于或等于0。所以，为了使$sqrt{1-x}$在实数范围内有意义，必须满足$1-x geq 0$。解这个不等式，将$x$移到不等式右边，得到$1 geq x$，即$x leq 1$。

# 题目 22
数学 填空题 难度: 4/5

🖼️ 题目附图
📐 图形描述
正比例函数和反比例函数图像的坐标系图
✓ 正确答案
$x > 2$ 或 $-2 < x < 0$
✗ 你的答案
$x e 0$
💡 详细解析
1. 求$k$的值： 由于正比例函数$y_1 = 2x$和反比例函数$y_2 = frac{k}{x}$的图像有一个交点的横坐标是2，将$x=2$代入$y_1 = 2x$，得到$y_1 = 2 imes 2 = 4$。所以交点坐标为$(2,4)$。 将交点坐标$(2,4)$代入反比例函数$y_2 = frac{k}{x}$，得到$4 = frac{k}{2}$，解得$k=8$。 所以反比例函数为$y_2 = frac{8}{x}$。 2. 解不等式$y_1 > y_2$： 即 $2x > frac{8}{x}$。 这个不等式需要分两种情况讨论，因为$x$可能为正或负： 情况1：当$x > 0$时（不等式两边同乘以$x$，不等号方向不变） $2x^2 > 8$ $x^2 > 4$ 由于$x > 0$，所以$x > 2$。 情况2：当$x < 0$时（不等式两边同乘以$x$，不等号方向改变） $2x^2 < 8$ $x^2 < 4$ 由于$x < 0$，所以$-2 < x < 0$。 综上所述，自变量$x$的取值范围是$x > 2$或$-2 < x < 0$。

# 题目 24
数学 填空题 难度: 4/5
如图，在平面直角坐标系中，反比例函数$y=frac{k}{x}$的图像与一次函数$y=x+1$的图像交于点A、B。若$S_{ riangle AOB}=2$，则$k$的值是\_\_\_\_。

📐 图形描述

✗ 你的答案
2
💡 详细解析
设点A的坐标为$(x_A, y_A)$，点B的坐标为$(x_B, y_B)$。 联立反比例函数和一次函数方程： $frac{k}{x} = x+1$ $k = x(x+1)$ 移项整理得二次方程：$x^2+x-k=0$。 点A、B的横坐标$x_A, x_B$是该方程的两个根。根据韦达定理： $x_A+x_B = -1$ $x_A x_B = -k$ 一次函数$y=x+1$与y轴的交点设为C。当$x=0$时，$y=0+1=1$，所以$C(0,1)$。 三角形$AOB$的面积可以通过将它分解为$ riangle AOC$和$ riangle BOC$的面积之和来计算（假设A、B分别位于y轴两侧，即$x_A$和$x_B$异号）。 $S_{ riangle AOB} = S_{ riangle AOC} + S_{ riangle BOC}$。 $ riangle AOC$以OC为底，高为$|x_A|$；$ riangle BOC$以OC为底，高为$|x_B|$。 $S_{ riangle AOB} = frac{1}{2} imes OC imes |x_A| + frac{1}{2} imes OC imes |x_B|$。 由于$OC=1$，且考虑到$x_A, x_B$异号（反比例函数$y=k/x$与直线$y=x+1$的交点通常位于y轴两侧，且由$x^2+x-k=0$可知，$x_A x_B = -k$，若$k>0$，则$x_A x_B < 0$，两根异号），不妨设$x_A > 0, x_B < 0$。 则$S_{ riangle AOB} = frac{1}{2} imes 1 imes x_A + frac{1}{2} imes 1 imes (-x_B) = frac{1}{2}(x_A - x_B)$。 已知$S_{ riangle AOB}=2$，所以$frac{1}{2}(x_A - x_B) = 2$，即$x_A - x_B = 4$。 现在我们有方程组： 1) $x_A + x_B = -1$ 2) $x_A - x_B = 4$ 将两方程相加：$(x_A+x_B) + (x_A-x_B) = -1+4 Rightarrow 2x_A = 3 Rightarrow x_A = frac{3}{2}$。 将$x_A = frac{3}{2}$代入方程1：$frac{3}{2} + x_B = -1 Rightarrow x_B = -1 - frac{3}{2} = -frac{5}{2}$。 最后根据韦达定理$x_A x_B = -k$，将$x_A, x_B$的值代入： $(frac{3}{2})(-frac{5}{2}) = -k$ $-frac{15}{4} = -k$ $k = frac{15}{4}$。
